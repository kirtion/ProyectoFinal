# < NXT Control Remoto >
This application lets you control Lego Mindstorms NXT robots.
An android phone sends the commands to another phone via WiFi, which sends them to the NXT via BlueTooth.

Controls:
  - Tilt
  - Pad
  - Touchpad

Text to speech funsies.
